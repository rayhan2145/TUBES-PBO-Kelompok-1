## Manajemen Gudang Inventaris - Tugas Besar PBO Kelompok 1

Sistem Manajemen Gudang Inventaris adalah sistem yang dibuat untuk membantu pihak manajemen gudang untuk dapat memonitor transaksi barang masuk atau barnag keluar, serta membantu pihak manajemen gudang untuk membuat laporan menggunakan cetak excel

Fitur utamanya antara lain adalah sebagai berikut.

1. fitur Autentikasi
2. fitur Transaksi Barang Masuk dan Keluar
3. fitur Pendataan Barang
4. fitur Cetak Laporan
5. fitur Filterisasi Barang dan Kategori.

### Arsitektur Sistem

- JSP
- Java Servlet

## Installasi

Untuk instalasi project ini, berikut adalah tahapannya:

1. Git clone project ini

```sh
    git clone https://github.com/rayhan2145/TUBES-PBO-Kelompok-1.git
```

2. Lalu masukkan library JAR yang terdapat pada folder [Dependencies](/dependencies/)

3. Jalankan pada Netbeans
